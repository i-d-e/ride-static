# Review Criteria

The reviews on this site are written against the IDE criteria for
reviewing digital editions. Four criteria sets cover different edition
types; each review states which set was applied and answers each
criterion as Yes, No, or Not Applicable.

The full criteria document is hosted by IDE; this page summarises the
set the editors applied to a given review and links to the canonical
criteria text.
